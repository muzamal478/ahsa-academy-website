# AHSA Academy & Publishers Website

A modern, responsive and SEO-optimized educational academy website developed for **Prof Anwar Hussain Syed Academy & Publishers (AHSA Academy)** located in Gujrat, Pakistan.

---

## 🌐 About The Project

AHSA Academy & Publishers is an educational institute providing:

- MDCAT Preparation
- ECAT Preparation
- IELTS & TOEFL Classes
- O Level & A Level Coaching
- FSC / ICS / FA Coaching
- BS University Courses
- Online Classes
- Home Tuition

This website was designed to create a professional online presence for the academy with a modern and user-friendly interface.
